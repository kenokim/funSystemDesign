# 8장 URL 단축키 설계

- 리다이렉션 종류
	- 301 Permanently Move: 영구적으로 Location 헤더 주소로 이동됨. 브라우저에서 캐시됨 -> 서버 부하 감소
	- 302 Found: 일시적으로: 일시적으로 Location 헤더 주소로 이동됨. 캐시X -> 통계에 유리
- 충돌방지
	- 해시 후 충돌 해소
